Polix.js

Annotations and Plugin Web Framework for Node.js

Get Started →

annotations

Enjoy the annotated development experience, no longer maintaining one or more routing files separately.

Plugin

Each lib no longer relies on the framework, injecting plug-ins into the framework to demonstrate project configurability.

Performant

Let's not talk about this.

# Get Statered

# install
npm i polix-cli -g

# init
pol init example

# build
make build

# start dev
make dev

# start pm2
make start

COMPATIBILITY NOTE

Polix.js requires Node.js >= 8.
